# NurseTatto by Lajami

Portafolio web profesional para **NurseTatto**, estudio de tatuajes en Guadalajara, Jalisco.

## 🖤 Stack

- [Astro](https://astro.build/) - Framework estático
- [Tailwind CSS](https://tailwindcss.com/) - Estilos

## ✨ Características

- Diseño responsive (mobile-first)
- Dark theme con paleta Gothic Dark Academia
- Galería de trabajos reales
- Formulario de contacto integrado con WhatsApp
- FAQ con acordeón
- Sección de Instagram
- Animaciones suaves

## 🚀 Ejecutar localmente

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```
